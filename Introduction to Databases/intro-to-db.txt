//ASSESSMENT ANSWER//
1.SQL vs NoSQL:>
    SQL-fixed schema, tables
    NoSQL-flexible schema, JSON-like data.
2.CAP Theorem:>
    3 properties: Consistency, Availability, Partition Tolerance
    Cannot achieve all 3 at the same time
3.When to use MongoDB:>
    Dynamic or changing data
    Need fast scaling
    Real-time applications
4.Why MongoDB uses BSON:>
    Faster than JSON
    Supports more data types (Date, Binary, etc.)
    Efficient storage
5.MongoDB Query:>
    db.students.find({gpa:{$gt:3.5},course:"CS101"})
